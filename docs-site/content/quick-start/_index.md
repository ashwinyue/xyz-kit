# 快速上手

本指南将帮助你快速了解 xyz-kit 的基本使用方法。

## 启动应用

安装完成后，应用会自动最小化到系统托盘。你可以看到：

- **macOS**: 菜单栏右上角的图标
- **Windows**: 任务栏右下角的系统托盘图标

## 唤醒窗口

使用全局快捷键唤醒应用窗口：

- **macOS**: `Cmd + Shift + F`
- **Windows**: `Ctrl + Shift + F`

窗口会出现在屏幕中央，并自动加载剪贴板内容。

## 基本工作流程

### 1. 准备数据

复制需要处理的文本到剪贴板。例如：

```
1,2,3,4,5
```

### 2. 唤醒窗口

按下快捷键 `Cmd/Ctrl + Shift + F`，窗口会自动显示并加载剪贴板内容。

### 3. 选择功能

点击右侧的功能图标，或使用默认功能。

### 4. 处理文本

按下 `Cmd/Ctrl + Enter` 或点击右下角的 "Enter" 按钮。

### 5. 获取结果

处理结果会自动复制到剪贴板，你可以直接粘贴使用。

## ID 格式转换示例

### 示例 1: 逗号分隔 → 引号格式

**输入**:
```
1,2,3,4,5
```

**输出**:
```
"1","2","3","4","5"
```

### 示例 2: 引号格式 → 正则表达式

**输入**:
```
"1","2","3","4","5"
```

**输出**:
```
^1$|^2$|^3$|^4$|^5$
```

### 示例 3: 正则表达式 → 逗号分隔

**输入**:
```
^1$|^2$|^3$|^4$|^5$
```

**输出**:
```
1,2,3,4,5
```

### 示例 4: 多行文本 → 引号格式

**输入**:
```
1
2
3
4
5
```

**输出**:
```
"1","2","3","4","5"
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + Shift + F` | 唤醒/显示窗口 |
| `Cmd/Ctrl + Enter` | 处理文本 |
| `Escape` | 隐藏窗口 |

## 使用技巧

### 1. 快速处理

选中功能图标后会立即执行处理，无需再按 Enter。

### 2. 自动复制

处理结果会自动复制到剪贴板，隐藏窗口后可以直接粘贴。

### 3. 失焦隐藏

点击窗口外部或切换到其他应用，窗口会自动隐藏。

### 4. 文本编辑

在处理前，你可以直接在窗口中编辑文本内容。

## 常见使用场景

### 场景 1: 数据库查询

将 ID 列表转换为 SQL 查询条件：

```sql
-- 原始数据
1,2,3,4,5

-- 转换为引号格式后
SELECT * FROM users WHERE id IN ("1","2","3","4","5");
```

### 场景 2: 日志分析

将 ID 列表转换为正则表达式用于日志过滤：

```
-- 原始数据
user123,user456,user789

-- 转换为正则表达式
^user123$|^user456$|^user789$
```

### 场景 3: 数据清洗

将多行数据转换为逗号分隔的单行：

```
-- 原始数据（多行）
apple
banana
orange

-- 转换为引号格式
"apple","banana","orange"
```
