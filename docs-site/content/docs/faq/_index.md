# 常见问题

## 安装相关

### Q: macOS 提示"无法打开，因为它来自身份不明的开发者"

**A**: 这是因为应用未经过 Apple 公证。解决方法：

1. 右键点击应用，选择"打开"
2. 或在终端运行：
   ```bash
   xattr -cr /Applications/xyz-kit.app
   ```

### Q: Windows Defender 阻止安装

**A**: 点击"更多信息" → "仍要运行"即可继续安装。

### Q: 如何卸载应用？

**A**: 
- **macOS**: 将应用拖入废纸篓
- **Windows**: 在"设置" > "应用"中卸载

## 使用相关

### Q: 快捷键不生效怎么办？

**A**: 
1. **macOS**: 检查是否授予了辅助功能权限（系统设置 > 隐私与安全性 > 辅助功能）
2. **Windows**: 检查是否有其他应用占用了 `Ctrl + Shift + F` 快捷键
3. 尝试重启应用

### Q: 窗口无法唤醒

**A**: 
1. 检查应用是否在系统托盘中运行
2. 尝试点击托盘图标手动显示窗口
3. 重启应用

### Q: 处理结果没有复制到剪贴板

**A**: 
1. 确保应用有剪贴板访问权限
2. 检查是否有其他应用占用剪贴板
3. 手动选中文本后复制

### Q: 如何更改快捷键？

**A**: 目前版本不支持自定义快捷键，默认为：
- macOS: `Cmd + Shift + F`
- Windows: `Ctrl + Shift + F`

### Q: 支持哪些文本格式？

**A**: 目前支持以下格式转换：
- 逗号分隔 (1,2,3)
- 引号格式 ("1","2","3")
- 正则表达式 (^1$|^2$|^3$)
- 多行文本

## 功能相关

### Q: 如何处理包含逗号的数据？

**A**: 如果数据本身包含逗号，建议先转换为多行格式，每行一个数据项。

### Q: 支持批量处理吗？

**A**: 支持。将所有数据放在一起，应用会自动识别格式并转换。

### Q: 处理大量数据会卡顿吗？

**A**: 应用针对性能进行了优化，可以流畅处理数千条数据。如果遇到卡顿，建议分批处理。

### Q: 能否保存处理历史？

**A**: 当前版本不支持历史记录功能，每次处理都是独立的。

## 技术相关

### Q: 应用使用什么技术栈？

**A**: 
- 前端: React + Vite + Tailwind CSS
- 后端: Rust + Tauri
- 状态管理: Zustand

### Q: 应用会收集用户数据吗？

**A**: 不会。应用完全在本地运行，不会上传任何数据到服务器。

### Q: 应用占用多少资源？

**A**: 
- 内存: 约 50-100 MB
- 磁盘: 约 10-20 MB
- CPU: 空闲时几乎为 0

### Q: 支持 Linux 吗？

**A**: 目前仅支持 macOS 和 Windows。Linux 支持在计划中。

## 更新相关

### Q: 如何检查更新？

**A**: 访问 [Releases 页面](https://github.com/ashwinyue/xyz-kit/releases/latest) 查看最新版本。

### Q: 如何更新应用？

**A**: 下载最新版本的安装包，重新安装即可。

### Q: 更新会丢失配置吗？

**A**: 不会。配置文件保存在用户目录中，更新不会影响。

## 故障排除

### Q: 应用无法启动

**A**: 
1. 检查系统是否满足最低要求
2. 查看系统日志中的错误信息
3. 尝试重新安装
4. 如果问题持续，请[提交 Issue](https://github.com/ashwinyue/xyz-kit/issues)

### Q: 窗口显示异常

**A**: 
1. 尝试重启应用
2. 检查显示器分辨率和缩放设置
3. 更新显卡驱动

### Q: 快捷键冲突

**A**: 
1. 检查其他应用的快捷键设置
2. 临时禁用可能冲突的应用
3. 等待后续版本支持自定义快捷键
