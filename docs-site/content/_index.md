# 欢迎使用 xyz-kit

xyz-kit 是一个轻量级的桌面文本处理工具，专为开发者设计，提供快速的 ID 格式转换功能。

## ✨ 特性

- 🚀 **全局快捷键唤醒** - `Cmd+Shift+F` (macOS) / `Ctrl+Shift+F` (Windows)
- 📋 **自动读取剪贴板** - 唤醒时自动加载剪贴板内容
- 🔄 **智能格式转换** - 自动识别并转换 ID 列表格式
- ⚡ **即时处理** - 处理结果自动复制到剪贴板
- 🎯 **轻量级** - 常驻系统托盘，不占用任务栏
- 🌐 **跨平台** - 支持 macOS、Windows
